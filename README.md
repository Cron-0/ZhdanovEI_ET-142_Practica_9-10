# ЖдановЕИ_ЕТ-142
Вариант 9

1. Структуры, массивы структур

"Определить структурный тип, описывающий маркетплейсы (название маркетплейса, количество ПВЗ по стране, город дислокации, рыночная стоимость, тип маркетплейса: продуктовый, товары для дома, все-подряд; тип бизнеса компании: основной, вспомогательный, франшиза и др.). Заполнить структурный массив 20-ю записями. Переписать из исходного массива в другой массив, информацию только о продуктовых маркетплейсах. Затем новый массив отсортировать по количеству ПВЗ. Вывести все данные по конкретному маркетплейсу. Вывести 3 самых популярных маркетплейса. Реализовать функцию изменения данных о маркетплейсе. В отдельный массив поместить все маркетплейсы не из Москвы. Реализовать вывод отфильтрованных данных в виде оберточной функции."

Для задания 3 использованы следующие входные данные:

	Введите название для изменения: METRO
	Новое название: Лемана_ПРО
	Новое кол-во ПВЗ: 120
	Новый город: Тюмень
	Новая стоимость: 75,5
	Тип (0-продукт,1-дом,2-все): 1
	Бизнес (0-основа,1-сторонний,2-франшиза): 2
  
Вывод:

	Успешно! Новая информация:
	Название: Лемана_ПРО
	ПВЗ: 120
	Город: Тюмень
	Стоимость: 75,50 млн
	Тип: Товары для дома
	Бизнес: Франшиза

2. Работа с файлами

Пример:

	Введите данные (название ПВЗ, -1 для завершения):
